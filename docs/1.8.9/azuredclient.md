# AzuredClient Alternatives

ChatTriggers may cause many performance issues depending on your modules and most of the modules you can get are readily available in other mods. This page is dedicated to the AzuredClient ChatTriggers module.

List is a WIP but should have all features just is missing some alternatives. Feel free to make a pull request to add them.
